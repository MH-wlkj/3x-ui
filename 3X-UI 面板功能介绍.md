## 2. 入站管理

创建和管理代理入站（即代理端口），每个入站对应一种协议和端口。

**支持的协议：**
VMess、VLESS、Trojan、Shadowsocks、Hysteria、Hysteria2、WireGuard、HTTP、SOCKS5、Mixed、Tunnel

**使用方法：**

1. 点击「添加入站」按钮
2. 选择协议类型（如 VMess、VLESS、Trojan 等）
3. 设置端口、备注、传输协议（TCP/WS/gRPC/QUIC 等）
4. 配置 TLS/Reality 加密
5. 保存后入站自动生效

**入站列表功能：**
- 查看所有入站及其运行状态
- 编辑、删除、启停入站
- 查看入站流量统计
- 配置入站的客户端流量限制和到期时间

## 3. 客户端管理

管理连接到入站的客户端，是日常使用最频繁的页面。

**功能列表：**

| 功能 | 说明 |
|------|------|
| 客户端列表 | 表格显示所有客户端，含邮箱、状态、流量、速率、到期时间 |
| 添加客户端 | 支持单个添加和批量添加 |
| 编辑客户端 | 修改邮箱、流量限额、到期时间、TG ID、备注等 |
| 启用/禁用 | 单个或批量开关客户端 |
| 流量重置 | 重置客户端流量计数 |
| 删除客户端 | 单个删除或批量删除 |
| 批量调整 | 批量延长到期时间、增减流量、设置 flow |
| 分组管理 | 将客户端加入分组，按组筛选 |
| 附加入站 | 将客户端附加到多个入站 |
| **导出二维码** | 选中客户端 → 更多 → 批量导出二维码，导出含二维码的 HTML/XLSX |
| **订阅链接** | 选中客户端 → 更多 → 订阅链接，批量复制订阅 URL |
| 导入客户端 | 通过 JSON 批量导入客户端 |
| 筛选和排序 | 按状态、协议、入站、到期时间等筛选和排序 |

**批量导出二维码使用方法：**

1. 勾选需要导出的客户端（可多选）
2. 点击「More」→「Batch QR export」
3. 弹窗自动加载客户端链接
4. 可选择：
   - **Copy links**：复制所有链接到剪贴板
   - **Export XLSX**：导出 Excel 表格（名字 | 链接 | 二维码）
   - **Export HTML**：导出独立 HTML 页面（含二维码图片）

## 4. 客户端生成器

批量生成 SOCKS5 上游节点 → Xray 客户端 + 出站规则 + 路由规则 的一键工具。

**使用方法：**

1. **填写生成配置**
   - Email 前缀/后缀：控制生成的客户端邮箱命名
   - 流量限制：每个客户端的流量上限
   - 目标入站：选择客户端要附加到的入站
   - 命名规则：使用 IP 命名或顺序数字命名
   - 出站协议：SOCKS5 或 HTTP

2. **输入上游节点列表**
   - 格式：`备注:IP:端口:账号:密码` 或 `IP:端口:账号:密码`
   - 输入后自动解析并显示预览表格

3. **点击「一键生成」**
   - 自动生成客户端 JSON、出站规则、路由规则
   - 生成后显示预览表格，可查看每个客户端的详细信息

4. **部署方式**
   - **一键部署到面板**：自动通过 API 创建客户端 + 添加出站/路由规则 + 重载 Xray
   - **手动复制**：分别复制客户端 JSON、出站规则、路由规则到对应位置

## 5. Xray 配置

编辑 Xray 核心的完整配置文件，包含以下标签页：

### 5.1 基础配置 (Basic)

**功能说明：**
- 查看和编辑 Xray 核心的基础配置参数
- 配置日志级别、日志路径
- 配置出站测试 URL（用于测试出站连通性）

**使用方法：**
直接在编辑器中修改配置项，点击保存后生效。

> **截图位置：**
>
>

### 5.2 路由规则 (Routing)

**功能说明：**
- 管理 Xray 的路由规则，控制流量转发策略
- 支持按域名、IP、端口、协议、用户等多种条件路由

**使用方法：**
1. 点击「添加规则」创建新路由规则
2. 设置匹配条件（域名、IP、端口、入站标签等）
3. 指定目标出站或均衡器
4. 支持拖拽排序

> **截图位置：**
>
>

### 5.3 出站管理 (Outbounds)

**功能说明：**
- 管理所有出站连接配置
- 支持 SOCKS5、HTTP、Direct、Block、DNS 等协议
- 支持 Warp、NordVPN 快速集成
- 支持订阅出站（自动从 URL 拉取出站列表）

**使用方法：**
1. 点击「Outbounds」添加新的出站
2. 设置出站协议、目标地址、端口
3. 支持批量删除（勾选多行 → 删除）
4. 支持导入/导出出站配置
5. 支持测试出站连通性（TCP/HTTP/Real delay）

> **截图位置：**

### 5.4 均衡器 (Balancers)

**功能说明：**
- 配置多个出站之间的负载均衡
- 支持随机、轮询、LeastLoad、LeastPing 等策略

**使用方法：**
1. 添加均衡器，设置名称
2. 选择要参与均衡的出站标签
3. 选择均衡策略
